## 项目介绍

「毛孩子邻里」是一个基于小区邻里的宠物服务平台，帮助宠主就近找到可信的帮手提供遛狗、喂猫、陪玩等上门服务。同一微信账号兼具宠主和帮手双重身份，需分开下载使用两个小程序。

### 账号与认证体系

- **宠主**：微信一键登录，创建服务需求
- **帮手**：需通过3步认证审核（身份+经验+小区），认证通过后在帮手端接单
- **抢单模式**：订单进入公开池，帮手先到先得
- **派单模式**：紧急单/高价值单由平台派给高评分帮手

## 功能模块

### 宠主端小程序
- 首页（快捷入口、进行中订单、附近帮手）
- 发现（搜索筛选帮手、帮手详情）
- 发布需求（服务类型/宠物/时间/地址/预算）
- 订单管理（进行中/已完成，含服务凭证查看）
- 个人中心（宠物/钱包/优惠券/地址/消息）

### 帮手端小程序
- 首页（今日任务/平台派单/抢单池）
- 订单管理（进行中/已完成）
- 收入中心（可提现余额/结算周期/提现记录）
- 个人中心（帮手认证/等级勋章/服务时间设置）

### 后台管理系统
- 工作台（运营数据概览、图表、待办事项）
- 用户/帮手/宠物/小区管理
- 订单管理、紧急求助、提现审批
- 财务管理（平台抽成、收支流水）
- 优惠券管理（创建/发放/核销）
- 评价管理、内容社区、投诉管理
- 消息推送、认证审核、系统设置
