# 毛孩子邻里 - lpet-prototype

宠物服务邻里互助平台原型

## 在线访问

- **小程序原型**: https://jiyankang.github.io/lpet-prototype/index.html
- **后台管理原型**: https://jiyankang.github.io/lpet-prototype/admin.html

## 项目介绍

「毛孩子邻里」是一个基于小区邻里的宠物服务平台，帮助宠主就近找到可信的帮手提供遛狗、喂猫、陪玩等上门服务。

## 功能模块

### 小程序端
- 首页（服务入口、热门帮手）
- 发现页（搜索帮手、分类筛选）
- 发布需求（选择服务、宠物、时间）
- 订单管理（进行中/已完成订单）
- 个人中心（宠主/帮手角色切换）

### 后台管理
- 工作台（运营数据概览）
- 用户/帮手/宠物/小区管理
- 订单管理、紧急求助、提现审批
- 优惠券管理、评价管理、内容社区
- 消息推送、认证审核、投诉管理
